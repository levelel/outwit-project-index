<h1 align="center">几维 Outwit 项目一览</h1>

几维（Outwit）输入法是面向专业输入体验打造的下一代输入法，提供跨平台的极简配置、实时主题预览以及可编排的智能功能。该仓库作为 Outwit 生态的索引页，帮助你快速定位官方站点与开源项目。

## 资源导航

- **官方网站**：<https://outwitapp.com>
- **主题设计器入口**：中国 <https://designer.outwitapp.com> 海外 <https://designer.outwit.app>
